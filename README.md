#  Mood to Music Recommender

## Project Description

The Mood to Music Recommender is a web application that suggests songs based on the user's mood and preferred language. The system dynamically fetches songs using the YouTube API and provides an interactive and responsive user interface.

---

##  Features

*  Mood selection with interactive cards
*  Language-based filtering (Telugu, Hindi, English, Tamil)
*  Dynamic song recommendations using YouTube API
*  Fully responsive design (mobile + laptop)
*  Hover effects and smooth UI interactions

---

##  Technologies Used

* Frontend: HTML5, Bootstrap, CSS
* Scripting: JavaScript, jQuery
* Backend: Flask (Python)
* API: YouTube Data API v3
* Version Control: Git & GitHub
